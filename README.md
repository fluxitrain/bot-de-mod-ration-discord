# Discord Moderation Bot

This repository contains a simple moderation bot for Discord. It includes the basic setup and a moderation cog.

## Setup
1. Install the required packages using `pip install -r requirements.txt`
2. Create a `config.py` file with your bot token and prefix.
3. Run the bot using `python main.py`.

## Features
- Basic moderation commands
- Easy to extend

Feel free to contribute!
